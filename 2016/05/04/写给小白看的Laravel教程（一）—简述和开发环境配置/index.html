写给小白看的Laravel教程（一）— 简述和开发环境配置

  • Laravel
  • PHP
  • Web

写了许久的 Java ，在一些需要一个人完全独自完成的快速开发的项目中，我选择了 PHP ，用纯 PHP 完成一个CMS以后，我还是觉得很累 ，而且代码写的很不优雅，然后寻找到了ThinkPHP ，看过开发者的访谈 ，以前也是做 Java 的，在ThinkPHP中我看到了很多Java的影子，写的很舒服 ，用TP写完了一个BBS，小型ERP以后觉得TP还缺点什么？
然后我遇到了Laravel ，让PHP不一样的活法。
此教程以Lravel5.1为基础，为php新手和Laravel学习困难户详细教程，当然你html，css ，js ，php基础得会。
为什么选择Laravel5.1?

Laravel 5.1 是 LTS 的发行版本, 发行于 2015 年 6 月份，自发布起提供两年时间的 Bug 修复（2017 年 6 月）, 3 年时间的安全修复（2018 年 6 月）。

安装homestead

homestead 可以说是Laravel的第一个拦路虎，有人是composer，我们先来说说homestead的安装。
前置条件安装Vagrant,virtualbox,这个两个安装很简单，这里就不说了。
要完成homestead环境的安装，需要完成三步，一是vagrant添加homestead包，第二步是配置Homestead ,第三步启动虚拟机

一　添加homestead包

如果你有很屌的宽带，你可以直接使用

大部分人都达不到这个速度，终端下载又很坑，所以我们选择用迅雷下载，先去https://atlas.hashicorp.com/laravel/boxes/homestead选择适合的版本下载，一般建议下载最新的版本.
下载的方法就是选择某个版本后，然后在路径增加providers/virtualbox.box 列如：https://atlas.hashicorp.com/laravel/boxes/homestead/versions/0.4.4/providers/virtualbox.box,然后迅雷下载不表。下载后将box放在~/boxs
